- +lsp ::   
  Enable LSP support for ~vhdl-mode~  and ~vhdl-ts-mode~.

* Configuration
This module has 2 variables to set the language server:
- ~+vhdl-lsp-server~
- ~+vhdl-lsp-server-path~

~+vhdl-lsp-server~ can take 4 values ~'vhdl-tool~, ~'hdl-checker~, ~'vhdl-ls~, and ~'ghdl-ls~.
This set the language server. 
Its default value is ~'vhdl-tool~.

~+vhdl-lsp-server-path~ takes president over ~+vhdl-lsp-server~. 
It is to set the path to a language server. Its default value is ~nil~.

An example of a configuration could be.
#+begin_src emacs-lisp
(setq +vhdl-lsp-server 'vhdl-ls
      +vhdl-lsp-server-path nil)
#+end_src
