* Description :unfold:
This module adds support for the [[https://odin-lang.org/][Odin]] programming language.

- Syntax highlighting (with or without tree-sitter)
- Auto-format (with ~odinfmt~, requires [[doom-module::editor format]])
- LSP integration (via ~ols~)

** Module flags
# Flags should be in alphanumerical order.
- +lsp ::
  Enable LSP support for ~odin-mode~ and ~odin-ts-mode~. Requires [[doom-module::tools
  lsp]] and an LSP server (e.g. =ols=).
- +tree-sitter ::
  Leverages tree-sitter for better syntax highlighting and structural text
  editing. Requires [[doom-module::tools tree-sitter]]. Emacs 30 or newer is
  recommended.

* Installation
[[id:01cffea4-3329-45e2-a892-95a384ab2338][Enable this module in your ~doom!~ block.]]

This module has no hard requirements for basic syntax highlighting, but beyond
that, benefits from having the following executables in your ~$PATH~:
- ~odin~
- ~odinfmt~ (for the [[doom-module::tools format]] module)
- ~ols~ (for [[doom-module:+lsp]])
