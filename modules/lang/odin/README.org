* Description :unfold:

To get started with Odin, you need a functional installation of [[https://odin-lang.org][Odin]], with optional LSP support for LSP via [[https://github.com/DanielGavin/ols][OLS]].

- Syntax highlighting (~odinfmt~)
- Syntax checking (~ols~)
- Code completion and LSP integration (~ols~)

* Installation
[[][Enable this module in your ~doom!~ block.]]

To get started with Odin, you need the Odin tool. You can download from [https://odin-lang.org/docs/install/#official-releases] and add to $PATH or from [[https://odin-lang.org/docs/install/#package-managers][System Package Manager]].

~ols~ is the language server for Odin, which provides:
- Completion
- Go to definition
- Semantic tokens
- Document symbols
- Rename
- References
- Signature help
- Hover

** LSP support

This module supports LSP integration. For it to work you’ll need:

 1. ~ols~ is downloaded
 2. ~ols~ is compiled
 3. ~ols~ is in $PATH
 4. The [[doom-module::tools lsp]]
